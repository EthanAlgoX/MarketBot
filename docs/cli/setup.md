# `marketbot setup`

Initialize `~/.marketbot/marketbot.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
marketbot setup
marketbot setup --workspace ~/.marketbot/workspace
```

To run the wizard via setup:

```bash
marketbot setup --wizard
```
