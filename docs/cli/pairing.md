# `marketbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
marketbot pairing list whatsapp
marketbot pairing approve whatsapp <code> --notify
```
